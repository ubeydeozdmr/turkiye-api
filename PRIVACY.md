# Privacy Policy

Last updated: 2026-06-13

TurkiyeAPI is an independent open API project operated by Ubeyde Emir Özdemir. It provides public Türkiye administrative-division data through `https://api.turkiyeapi.dev`.

TurkiyeAPI is not affiliated with, endorsed by, or operated by any official government institution.

## Data You Send to the API

The API is public and does not require an account. Requests may include query parameters, headers, an optional `x-api-key`, or an `authorization` header if a client chooses to send them.

Do not send secrets, private personal data, or confidential business data in API query parameters.

## Application Logs

Application logs are minimized and used for operations, debugging, abuse prevention, and rate-limit visibility.

Application logs may include:

- Request ID.
- API version.
- HTTP method.
- Request path.
- Matched route.
- Query parameter names.
- Status code.
- Response time.
- Cache status.
- Rate-limit summary.
- Structured error code and message.

Application logs do not intentionally include:

- Request bodies.
- Response bodies.
- Cookies.
- Authorization headers.
- API keys.
- Full request headers.
- Full query parameter values.

## Server Access Logs

Production server or reverse-proxy access logs may include IP address, user agent, request path or URI, timestamp, status code, response size, and response time.

These logs are used for security, abuse prevention, rate-limit enforcement, traffic troubleshooting, and service reliability. Production access logs are intended to be retained for a limited period, typically up to 14 days, unless longer retention is needed to investigate abuse, security incidents, or operational problems.

## Hosting and Processors

The production service may be hosted on infrastructure provided by Hetzner Online GmbH in Germany or the European Union.

Infrastructure providers may process limited technical data, such as IP addresses and server logs, as needed to provide hosting, networking, and security services.

## Cookies and Analytics

The API itself does not use browser cookies.

TurkiyeAPI does not intentionally use advertising pixels or third-party marketing trackers for the API service. If the website or documentation adds analytics or cookies in the future, this policy should be updated before those tools are enabled.

## Sharing

Operational logs are not sold. They may be shared only when necessary with infrastructure providers, security responders, legal authorities where required, or maintainers who need access to operate and protect the service.

## Your Requests

For privacy questions, access requests, deletion requests, or objections, email `ubeydeozdmr@gmail.com`.

Some technical logs may not be tied to a directly identifiable account because TurkiyeAPI does not require accounts. Requests will be handled based on the information reasonably available.

## Changes

This policy may change as the project, infrastructure, or logging practices change. Material changes should be reflected in this file and linked documentation.
