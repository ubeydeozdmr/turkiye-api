# Support

TurkiyeAPI is free and public. Support is best-effort.

## Getting Help

- Read the [README](./README.md).
- Check the [documentation](https://docs.turkiyeapi.dev/tr/v2/guide/).
- Search existing [GitHub issues](https://github.com/ubeydeozdmr/turkiye-api/issues).
- Open a new issue for reproducible bugs, documentation problems, or data corrections.

For security issues, do not open a public issue. Follow [SECURITY.md](./SECURITY.md).

## Supporting the Project

If TurkiyeAPI helps your project, you can support hosting and maintenance costs through the GitHub sponsor button configured in [.github/FUNDING.yml](./.github/FUNDING.yml).

Support is optional and does not change API access, rate limits, or issue priority.
